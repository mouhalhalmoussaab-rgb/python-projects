# Analyse Météo Ajaccio 2014

Ce projet consiste à analyser les relevés météorologiques de la station d’Ajaccio pour l’année 2014.  

**Objectifs :**  
- Traitement et nettoyage des données météo avec Python et Pandas  
- Calcul de statistiques simples : températures moyennes, précipitations, neige, vitesse du vent  
- Visualisation des données et comparaison avec les normales climatiques (1981–2010)  
- Analyse de la direction du vent par quadrant

**Technologies :** Python, Pandas, Matplotlib, Seaborn

**Structure :**  
- `data/` : fichiers CSV  
- `notebooks/` : notebook Jupyter pour l’analyse  
- `scripts/` : scripts Python  
- `results/` : graphiques et résultats
